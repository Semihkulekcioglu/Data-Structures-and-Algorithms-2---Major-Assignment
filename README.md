 <h2>Problems</h2>


Problem 1: Greedy Algorithm
Find the maximum number of meetings that can be accommodated in a meeting room. It's like Longest Increasing Subsequence

Problem 2: Greedy Algorithm
Given a set of jobs with deadlines and profits, find the number of jobs done and the maximum profit. It's like Longest Common Subsequence

Problem 3: Dynamic Programming
Repeat Problem 2 using dynamic programming. It's like Longest Increasing Contiguous Subarray

Problem 4: Greedy Algorithm
Find the longest chain of pairs that can be formed from a given set of pairs. It's like Array Rotation

Problem 5: Dynamic Programming
Repeat Problem 4 using dynamic programming. It's like Two Sum

Problem 6: Backtracking
Find all possible paths for a rat to reach its destination in a square matrix. It's like Trapping Rain Water

Problem 7: Backtracking
Determine if a graph can be colored with at most M colors such that no two adjacent vertices have the same color. It's like Maximum Subarray Sum

Problem 8: Backtracking
Find all possible combinations of valid IPv4 IP addresses. It's like Fibonacci Numbers

Problem 9: Backtracking
Find all possible words from a dictionary that can be formed by a sequence of adjacent characters on a board. It's like Fibonacci Sequence

Problem 10: Divide and Conquer
Find the element at the kth position of the final sorted array formed by merging two sorted arrays. It's like Binary Search

Problem 11: Divide and Conquer
Given N books and M students, find the permutation where the sum of the maximum number of pages allotted to a student is minimum. It's like Creating Special Sequences

Problem 12: Divide and Conquer
Find the total number of possible special sequences of numbers. It's like Sorting an Array

## Operating Instructions

Each problem has a separate C++ file with instructions at the beginning for solving the specific problem.

To solve any problem, follow these steps:

1-) Open the relevant C++ file.

2-) Compile the file: g++ Problem_1.cpp -o Problem_1.

3-) Run the resulting executable: ./Problem_1.
